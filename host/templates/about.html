AstroWatch
About Us

AstroWatch (BrowseWatch Lab) is a controlled cybersecurity research environment built to simulate and monitor browser-based activity across multiple clients. It supports secure agent deployment, real-time monitoring, and structured data collection from simulated endpoints.

This project is conducted under institutional authorization and designed for educational and research purposes only. All components, including the dashboard, agents, and data exfiltration logic, operate within sandboxed infrastructure for risk-free experimentation.

For more information, source code access, or lab configuration, contact the development lead or refer to the internal documentation repository.
